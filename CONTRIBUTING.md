# Contributing

Thank you for your interest in the Joe's Corner API specification.

## How the spec is maintained

The `openapi.json` and `openapi.yaml` files are auto-generated. They are overwritten on each export, so pull requests that modify these files directly will not be accepted.

## What you can do

**Open an issue** if you notice:

- A discrepancy between the spec and the live API behavior
- Missing or unclear documentation
- A feature request or suggestion for the API

**Submit a pull request** for changes to documentation files (`README.md`, `CONTRIBUTING.md`, `CHANGELOG.md`).

## Questions

For general questions about using the API, please open an issue.
